## Peercoin
Peercoin is a centralized peer to peer network designed to connect users that want to buy cryptocurrencies with those who sell cryptocurrencies.

## From source
* Install mongodb
* Install rabbitmq
* take out the .sample in the resources/ folder of manager and web

## run
* use `./gradlew bootrun` (or `.\gradlew.bat bootrun` for windows)
* add currencies and payment methods at localhost:8080
* use application at localhost:8081
